# Mobile Recipe & Resource Manager

A native Android application demonstrating local data persistence and dynamic UI rendering. This project showcases the integration of relational databases within a mobile environment.

## Technical Highlights
* **Local Persistence:** Utilizes **SQLite/Room Persistence Library** for robust, on-device data storage.
* **Component Architecture:** Implemented using a structured Activity/Fragment lifecycle to ensure state preservation during configuration changes.
* **Efficient UI:** Features a `RecyclerView` with custom Adapters for optimized memory usage during list scrolling.

## Tech Stack
* **Language:** Java / Kotlin
* **Database:** SQLite
* **UI:** XML Layouts with Material Design components.
